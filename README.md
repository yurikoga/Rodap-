# Responsive Footer using HTML and CSS

Um projeto simples de **footer (rodapé) responsivo** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **rodapé moderno e adaptável**, ideal para qualquer tipo de site.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Layout responsivo
- Organização de links
- Design simples e limpo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
